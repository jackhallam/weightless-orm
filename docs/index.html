weightless-orm

A lightweight database mapping library for Java

  • Quickstart
  • Github
  • Chat on Gitter

Weightless is an object-relational / object-document mapping library for Java. In other words, it helps you save Java objects to a database and query them later. Weightless currently supports MongoDB.
