weightless-orm

A lightweight database mapping library

Weightless is an object-relational / object-document mapping library for Java. In other words, it helps you save Java objects to a database and query them later. Weightless currently supports MongoDB.

Creating a database record and finding it later is a great place to start with the Weightless library. Press the green play button on the Repl.it below to begin the first example.

Find Multiple Database Records

Searching a database is extremely easy with Weightless because it conforms to the type you want to return. In this example we explore returning multiple records as well as sorting the results.
